语句块
```python
for i in [1, 2, 3]: 
    print(i)
```

语句行
```python
# 一个语句行对应一个文本行
print('Hello Python!') 
# 多个语句行对应一个文本行
print('Hello Python!');print('Hello Python!') 
# 一个语句行对应多个文本行
print( 
 'Hello Python!' + \ 
 'Hello World!' 
)
```
